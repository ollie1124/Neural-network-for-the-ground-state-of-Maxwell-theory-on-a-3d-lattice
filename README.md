# Equivariant-neural-networks-for-Maxwell-theory-in-3d
This project is an equivariant (i.e. U(1) gauge-invariant) convolutional neural networks (PyTorch) that model the ground states of 3d Maxwell theory on a square lattice. We implemented variational Monte Carlo (Metropolis-Hastings) for optimization, achieving lower energy estimates and faster convergence compared to traditional numerical methods.  This project is the outcome of the collaboration with Sriram Bharadwaj, Jack Isen and Aditya Prasad Dash.
  
Here is the overview of the files:
- "Paper report_NN for the ground state of 3d MW theory.pdf": We provided a comprehensive explanations including the code snippets.
- "Presentation slides_NN of ground state of 3d MW theory.pdf": Presentation slides that mention our motivation and brief explanations of the method and results.
- "Equivariant neural network for 3d Maxwell theory.ipynb": The project code.
